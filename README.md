## 📌 Project Overview

This project predicts the **FIFA World Cup 2026 Champion** using Machine Learning and Monte Carlo Simulation.

The model is trained on historical international football match data and team statistics to estimate match outcomes and simulate the complete FIFA World Cup tournament.

The project demonstrates an end-to-end sports analytics pipeline including:

- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Machine Learning Prediction
- Tournament Simulation
- Champion Probability Estimation

---

## 🎯 Objectives

- Predict match outcomes using Machine Learning
- Simulate the complete FIFA World Cup tournament
- Estimate each team's probability of becoming World Champion
- Visualize tournament progression and prediction results

---

The project uses historical international football match data containing:

- Team Names

- Goals Scored
